# Linked List

| Operation 	          | Big-O Time  	 |
|----------------------|---------------|
| Access i-th Element 	 | O ( n )  	    |
| Insert/Remove end 	  | O ( 1 )  	    |
| Insert/Remove middle 	 | O ( 1 )  	    |
* 하지만 중간에 삽입하거나 삭제할 때, 그 값을 찾는데 O ( n )
