# Sorting

| Sort 	                     | Big-O Time  	                      |
|----------------------------|------------------------------------|
| Bubble Sort	(stable)       | O ( n^2 )  	                       |
| Insertion Sort (stable)	   | O ( n^2 )  	                       |
| Selection Sort (unstable) 	 | O ( n^2 ) 	                        |
| Merge Sort (stable)        | O ( n log n )	                     |
| Quick Sort (unstable)      | O ( n^2 )	(avg. case O (n log n) ) |
| Bucket Sort (unstable)     | O ( n )                            |
